### ファイル読み込み
指定したファイル（file-sample.txt）をモード「r」（読み込みのみ）で開きます
```
// ファイルオープン
$fp = fopen("file-sample.txt", "r") or die("ファイルが開けないよぉ！");

```
### 読み込んだファイルを出力
読み込んだファイルを繰り返し処理「while」を使って、出力しています
```
// 読み込んだファイルを出力
while( !feof($fp) ) {
  echo fgets($fp);
}
```

```
<?php
// ファイルオープン
$fp = fopen("file-sample.txt", "r") or die("ファイルが開けないよぉ！");
 
// 読み込んだファイルを出力
while( !feof($fp) ) {
  echo fgets($fp);
}
 
// ファイルクローズ
fclose($fp);
?>

```
