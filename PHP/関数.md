### 関数の宣言と呼び出し
宣言
```
function 関数名( 引数 ) {
  // 処理
  return 戻り値;
}

```

呼び出し
```
変数 = 関数名( 引数 );
```

例　円の面積を計算する関数「circle」
```
<?php
// 関数の宣言
function circle($r) {
  return $r * $r * M_PI;
}
 
// 変数
$r = 3;
 
// 関数の呼び出し
$area = circle($r);
echo $area ."\n";
?>

```

### 関数の引数にデフォルト値を指定
```
function 関数名( 引数 = デフォルト値 ) {
  // 処理
  return 戻り値;
}

```

例
```
<?php
function piyo($a,$b=5){
  return $a*$b;
}
 
// 引数を1つ指定
echo piyo(3) ."\n";
// 引数を2つ指定
echo piyo(10,2) ."\n";
?>

```
